Simple widget plugin  to display youtube subscription functionality 

== Description ==
This widget plugin lets you add functionality  of youtube subscription

== Installation ==
1. Download the plugin
2. Upload the yotube-subscription folder to your site's wp-content/plugins
directory
3. Activate the plugin in the WordPress Admin Panel
4. Go to appearance->widgets and put your information there
